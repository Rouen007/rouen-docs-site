# 第一章：科学建仓的基本框架

> 建仓不是预测，而是在关键位置等待市场表态；确认后承担有限风险，失效后立即退出。

## 一、建仓不是预测，而是验证

日内交易不应先预测涨跌，再寻找理由。更稳健的过程是：

```text
提出假设
→ 等待价格到达关键位置
→ 观察市场反应
→ 确认假设成立
→ 承担有限风险
```

建仓前真正要回答的是：

> 市场是否已经提供了一个方向清晰、风险可定义、目标明确的机会？

## 二、建仓的五个要素

```text
位置 → 触发 → 确认 → 失效 → 目标
```

### 1. 位置

提前标记：

- 前高、前低
- 开盘区间
- VWAP
- 重要整数位
- 支撑与阻力
- 流动性或成交集中区域

位置只告诉我们“在哪里观察”，不直接告诉我们“应该买还是卖”。

### 2. 触发

常见触发包括：

- 突破关键阻力
- 跌破关键支撑
- 收回此前失守的位置
- 触碰边界后形成失败突破
- 高低点结构发生改变

触发代表市场开始表达方向，但不等于立即追单。

### 3. 确认

顺势建仓通常需要：

```text
突破关键位
→ 回踩突破位
→ 没有重新回到旧区间
→ 再次启动
```

反向建仓通常需要：

```text
触碰边界
→ 假突破或快速收回
→ 再次测试失败
→ 反向运行
```

第一根突破 K 线往往波动最大、滑点最差，因此回踩确认通常比第一时间追单更有利于控制风险。

## 三、图例一：突破后回踩确认的多头建仓

![历史复盘图例：收回关键区域后，回踩守住并继续上行](./assets/figure-1-breakout-retest.png)

*图 1：收回关键区域后，回踩守住并继续上行。图中价格仅用于历史复盘，不构成当前点位。*

图中蓝圈附近不是因为出现一根阳线就立即追多，而是先观察价格从低位收回关键区域；回踩时没有重新跌回弱势区间，随后再次上行并形成更高的高点和更高的低点。

```text
位置：前期低点和结构支撑区
触发：价格收回关键区域
确认：回踩不破，重新向上
失效：重新跌回收回区域
目标：上方下一个结构阻力区
```

第一次收回只能定义为“候选机会”；回踩守住并再次启动后，才具有更高质量。

## 四、图例二：突破失败后的空头建仓

![历史复盘图例：冲高失败、反抽不过并形成下行](./assets/figure-2-failed-breakout.png)

*图 2：冲高失败、反抽不过并形成下行。图中价格仅用于历史复盘，不构成当前点位。*

图中价格先向上测试高位，随后快速回落。之后价格反抽，但没有重新站回关键区域和 VWAP，最终持续形成更低的高点和更低的低点。

合理的空头逻辑不是看到长阴线就做空，而是等待突破被证明失败：

```text
位置：前高和上方阻力区
触发：冲高后快速收回
确认：反抽无法重新站回关键区域
失效：价格重新站回关键区域并回踩守住
目标：下方前低或下一个支撑区
```

## 五、失效点必须先于建仓确定

常见失效情况包括：

- 多头突破后重新跌回旧区间
- 空头跌破后重新站回关键位
- 回踩破坏原有结构
- 价格在预期时间内没有向目标移动
- VWAP 或关键位置重新被反向收回

建仓前必须写清：

```text
我为什么进场？
哪里证明我错了？
第一目标在哪里？
如果价格不动，多久退出？
```

如果只能说出“我觉得会涨”或“我觉得会跌”，说明交易仍处于预测阶段。

## 六、仓位应随证据增加

```text
只有位置：观察
位置 + 触发：试探或继续等待
位置 + 触发 + 回踩确认：建立主要仓位
确认 + 延续：才考虑增加
```

增加仓位必须发生在新的低风险确认点，而不是价格已经大幅远离关键位之后。

## 七、指标的证据权重

单一指标都可能失效：节点可能被直接突破，VWAP 可能被反复穿越，一笔大期权成交不一定代表新建仓，放量也可能是情绪释放。

更合理的观察顺序是：

```text
价格位置与结构
→ 突破或拒绝
→ 回踩确认
→ VWAP 与市场同步
→ 成交量和资金流
→ 其他辅助指标
```

辅助指标只能提高或降低信心，不能替代价格确认。

## 八、第一章执行清单

每笔交易前写下：

- 位置：
- 触发：
- 确认：
- 失效：
- 第一目标：
- 预计持仓时间：
- 最大风险：

执行规则：

1. 不在关键位置之间的中间区域主动开仓。
2. 不因为第一根大 K 线追单。
3. 不因为单一指标或单笔大单建仓。
4. 回踩没有确认时优先等待。
5. 仓位随着证据增加，而不是随着情绪增加。
6. 到达结构目标后分批兑现。
7. 失效后立即退出，不等待市场“证明自己”。

## 本章总结

科学建仓的本质是：

> 在关键位置等待市场表态，在回踩确认后承担有限风险，并在判断失效时退出。

```text
位置明确
→ 触发出现
→ 回踩确认
→ 小风险建仓
→ 结构延续再增加
→ 到目标分批兑现
→ 失效立即退出
```
