# 第四章：止损、风险与仓位设计

> 止损不是“我最多愿意亏多少钱”，而是市场在什么位置证明交易逻辑已经失效；仓位则负责把这个失效距离换算成可承受的金额。

## 一、先决定哪里错，再决定买多少

科学的下单顺序是：

```text
确定交易逻辑
→ 标记结构失效点
→ 计算入场到失效点的距离
→ 设定单笔风险金额
→ 倒推出允许仓位
→ 检查第一目标与盈亏比
→ 最后才下单
```

错误顺序通常是：

```text
先买一个习惯仓位
→ 发现波动太大
→ 为了少亏而随意收紧止损
→ 被正常波动扫出
→ 价格随后按原方向运行
```

因此，**止损距离决定仓位，仓位不能反过来决定止损位置**。

## 二、用 R 统一衡量风险

把每笔交易计划承担的最大亏损定义为 `1R`。

```text
1R = 账户权益 × 单笔风险比例
```

假设账户权益为 50,000 美元：

| 单笔风险比例 | 1R |
| --- | ---: |
| 0.25% | $125 |
| 0.40% | $200 |
| 0.50% | $250 |
| 1.00% | $500 |

风险比例不是越高越好。它只决定连续错误时账户回撤的速度，并不会提高策略本身的胜率。

同样是突破回踩后的③，如果入场位置距离下一供给区较远，潜在收益可能覆盖结构风险；如果已经贴近供给区，剩余空间不足，就不应追价。

### 学习阶段的示范参数

下面是一套用于训练的保守模板，不是适合所有账户的固定标准：

```text
单笔风险：0.25%—0.50%
单日硬上限：2R
连续两笔完整止损：当天停止主动交易
周度回撤达到 4R：下一周风险减半并复盘
```

只有在同一套 setup 积累足够样本、执行偏差稳定后，才讨论提高风险。不能因为连续盈利几次就临时放大仓位。

## 三、四种失效：价格、结构、时间与环境

### 1. 价格失效

价格越过一个明确数值，交易逻辑不再成立。

例如：

- 向上假突破做空，价格重新站上假突破高点。
- 跌破回抽做空，价格重新站回原支撑并完成回踩。
- 下方海龟汤做多，价格重新跌破扫荡低点。

价格失效最容易量化，但不能脱离 K 线收盘和市场波动机械执行。

### 2. 结构失效

价格行为已经破坏建仓依据，即使尚未触及最远止损，也应退出。

```text
多头：更高低点被破坏，旧区间被重新接受
空头：更低高点被突破，原阻力被重新收回
失败突破：市场重新接受在突破方向
节点反转：节点被连续实体穿越，不再产生拒绝
```

结构失效通常比“等到最大亏损”更早，因此它是主动退出依据；预设的最大止损则是执行失败或快速行情下的最后保护。

### 3. 时间失效

日内 setup 依赖价格在有限时间内离开建仓区。如果入场后持续停留在关键位附近，说明预期中的主动买卖力量没有出现。

训练时可以采用：

```text
1 分钟执行：入场后 2—4 根 K 仍无位移 → 减仓或退出
3 分钟执行：入场后 2—3 根 K 仍无位移 → 重新评估
0DTE：越接近到期，时间止损越不能放松
```

时间止损不是固定闹钟。强趋势可以较快确认；震荡环境中，迟迟不动本身就是 setup 质量下降的证据。

### 4. 环境失效

交易方向可能没有立即错误，但支撑它的环境已经改变：

- 关键节点或 GEX 结构快速迁移。
- 价格反复穿越 VWAP，原趋势退化为震荡。
- SPX 与 ES/NQ 明显不同步。
- 期权流与价格没有后续确认。
- 数据源异常、延迟或相互冲突。
- 重大事件即将公布，原波动假设失效。

环境失效时优先降仓、缩短目标或退出，而不是继续增加指标为持仓辩护。

### 建仓后的三层退出

建仓以后不能只保留一个很远的硬止损。更完整的退出顺序是：

```text
结构失效 → 主动退出
时间失效 → 减仓或退出
硬止损 → 处理快速行情和执行失败
```

硬止损是最后保护，不代表任何情况下都要等它触发。价格已经重新接受在旧区间，或者预期位移在规定时间内没有出现，都可以在硬止损之前结束交易。

| 路径 | 价格表现 | 管理动作 |
| --- | --- | --- |
| 影线测试 | 盘中短暂跌破，但实体重新收回关键位上方 | 暂时持有，不加仓，等待重新产生位移 |
| 结构失效 | 实体收进旧区间，反抽也无法重新站回 | 主动退出，不继续等待最远硬止损 |
| 时间失效 | 连续 3—4 根 1 分钟 K 仍围绕建仓区横盘 | 减仓或退出 |
| 盈利后新结构 | 先产生位移，再形成更高低点并重新突破 | 止损上移到新结构外侧，再考虑增加仓位 |

判断影线和实体时，可以采用：

```text
影线跌破、实体收回 → 观察反应，暂不认定完全失效
实体收回旧区间、再次反抽失败 → 结构失效
直接穿过最大风险线 → 执行硬止损，不等待收盘
```

盈利后也不能刚有浮盈就把止损推到成本价。应先等待“位移—形成新高低点—再次启动”完整出现，再依据新结构上移止损。

### 图解：影线扫损与结构失效

一根影线刺破后收盘收回，和实体收盘跌破、反抽失败，是两种不同证据。若保护性止损已经真实成交，应接受退出，之后等待新的③，而不是立即追回。

## 四、结构止损应该放在哪里

| Setup | 建议失效位置 | 不合理做法 |
| --- | --- | --- |
| 突破回踩做多 | 回踩低点下方，或重新进入旧区间的位置 | 紧贴突破线随意放一个固定点数 |
| 跌破回抽做空 | 反抽高点上方，或重新站回旧区间的位置 | 因期权价格波动就不断移动止损 |
| 上方海龟汤做空 | 扫荡高点外，或重新站稳区间上沿 | 看见第一根阴线后把止损放在阴线高点内 |
| 下方海龟汤做多 | 扫荡低点外，或重新跌回区间下沿 | 仅按账户亏损金额确定价格位置 |
| 区间边缘反转 | 边界外一格并结合拒绝 K 的极值 | 在区间中轴才止损 |

### 图解：止损应放在结构外

止损不能紧贴回踩低点，也不能因为仓位太大而缩短到结构内部。合理做法是先确定结构外的失效点，再根据距离缩小仓位。

<iframe src="./assets/stop-placement.html" title="止损位置与扫损对比" loading="lazy" style="width:100%;height:470px;border:0;display:block;"></iframe>

止损需要给正常噪音留出空间，但“更宽”不代表“更安全”。止损距离变宽时，仓位必须同步缩小。

## 五、仓位计算公式

### 1. 股票、ETF 或按点计价的产品

```text
单位风险 = |入场价 - 止损价| + 预估滑点与费用
允许数量 = 向下取整（1R ÷ 单位风险）
```

### 构造例：股票突破回踩

```text
账户权益：$40,000
单笔风险：0.25% = $100
计划入场：$52.00
结构止损：$51.50
预估滑点与费用：$0.05 / 股

单位风险 = 52.00 - 51.50 + 0.05 = $0.55
允许数量 = floor(100 ÷ 0.55) = 181 股
```

实际执行可进一步向下取整为 175 或 180 股，不能为了凑整数向上超过风险预算。

### 图解：固定风险下的仓位变化

在最大风险相同的情况下，止损距离从2点扩大到5点，允许数量就必须从50单位降到20单位。止损变宽并不代表不能交易，但必须同步减少仓位。

### 2. 期货或差价合约

```text
单位风险 = 止损点数 × 每点价值 + 滑点与费用
允许合约数 = 向下取整（1R ÷ 单位风险）
```

每个产品的点值和最小变动单位不同，必须使用交易所或券商的合约规格，不能凭股票仓位经验换算。

### 3. 买入期权

买入 call 或 put 时，到期最大亏损通常是支付的全部权利金；期权教育委员会也将买入 call 的最大亏损定义为初始权利金。[[OIC：Long Call](https://www.optionseducation.org/strategies/all-strategies/long-call)]

```text
绝对最大亏损 = 权利金 × 合约乘数 × 合约数
计划止损风险 ≈ |入场权利金 - 计划退出权利金| × 合约乘数
```

计划止损风险只能用于正常成交情形。快速波动、价差扩大或系统延迟可能使实际退出价差于计划，因此还要检查“全部权利金损失”是否会对账户造成不可接受的伤害。

SPX 期权的合约乘数为 100 美元，即权利金每变动 1.00，对单张合约价值影响 100 美元；SPX 采用现金结算和欧式行权。[[Cboe：SPX Options](https://www.cboe.com/tradable-products/sp-500/spx-options/)]

### 构造例：最小合约仍然太大

```text
1R：$200
一张 SPX 0DTE call 权利金：$3.20
购买成本：$320
计划止损：期权价格跌到 $2.20
正常计划风险：约 $100 + 滑点
极端最大风险：$320
```

虽然正常止损看似符合 1R，但一张合约的极端最大风险已经达到 1.6R。如果账户无法接受这种跳空或执行风险，正确动作不是放宽预算，而是：

1. 换用更小的产品；
2. 使用有限风险价差；
3. 等权利金下降且结构仍有效；
4. 放弃这笔交易。

**最小可交易单位超过风险预算时，不交易本身就是仓位管理。**

### 4. Debit Spread 等有限风险结构

以牛市 call 价差为例，其到期最大亏损是建立组合支付的净权利金；最大盈利为执行价宽度减去净权利金。[[OIC：Bull Call Spread](https://www.optionseducation.org/strategies/all-strategies/bull-call-spread-debit-call-spread)]

```text
净支出：$2.00
合约乘数：100
单组最大亏损：$200
```

如果 `1R = $200`，一组价差正好占用完整风险预算。但盘中仍应依据 SPX 结构失效主动退出，而不是因为“最大亏损有限”就持有到归零。

有限风险结构一旦建立，不能临时拆掉保护腿，把有限风险变成裸露风险。如果市场观点发生变化，应整体平仓，再重新建立新的独立交易。

## 六、分批建仓：证据增加，风险不能增加

分批建仓的正确逻辑是：

```text
候选位置出现 → 小仓试探
触发得到确认 → 增加主要仓位
价格再次延续 → 只在新止损可以上移时考虑增加
任何时刻的总计划亏损 ≤ 原定 1R
```

这里的 `1/4`、`3/4` 和“满仓”首先指风险预算的使用比例，不一定等于资金或合约数量的相同比例。交易工具的最小单位过大、无法合理分批时，应等待主要确认后一次建立符合风险预算的最小仓位，而不是为了凑出三段强行超额交易。

### 三段式示范

| 阶段 | 证据 | 仓位参考 | 风险要求 |
| --- | --- | ---: | --- |
| 观察仓 | 关键位置 + 初始触发 | 1/4 | 失效点必须明确 |
| 主仓 | 回踩守住 + 再次启动 | 增至 3/4 | 重新计算总风险，不得超过 1R |
| 延续仓 | 新结构形成且原仓止损可上移 | 最多满仓 | 不能在远离关键位时追价 |

这不是要求每笔交易都分三次。强趋势可能只给一次合理回踩；震荡环境则可能根本不值得建立主仓。

学习阶段可以先简化为两段：

```text
第一段：③（回踩结束后再次突破回踩结构的确认K）收线 + 微回踩守住 → 使用约 1/2—3/4 风险预算
第二段：形成新高低点 + 再次启动 → 原止损能够上移后才补足
```

如果第一段建仓后价格向失效点运行，证据正在减少，不能因为价格更便宜而增加仓位。只有交易已经产生位移、新结构降低了原仓风险时，才允许讨论延续加仓。

### 不能这样加仓

- 亏损后为了降低平均成本而加仓。
- 价格已经远离失效点，因为害怕错过而追仓。
- 新增仓位后，总风险超过原定 1R。
- 仅因期权流出现一笔大单就放大仓位。
- 组合已经是有限风险结构，却拆腿追求更高收益。

## 七、仓位应随环境折扣

基础仓位计算完成后，还要根据证据质量应用折扣：

| 环境 | 建议风险系数 | 原因 |
| --- | ---: | --- |
| 关键位 + 回踩确认 + 指数同步 | 1.0 | 主证据完整 |
| 价格确认，但 ES/NQ 不同步 | 0.5—0.75 | 跨市场证据冲突 |
| 开盘前半小时，GEX/flow 尚不稳定 | 0—0.5 | 容易出现假信号 |
| 价格在两个节点中间 | 0 | 无结构优势 |
| 数据源异常或事件即将公布 | 0 | 风险无法可靠估计 |
| 当日已亏 1R | 下一笔减半或停止 | 防止情绪和环境共同恶化 |

这里的系数是训练模板。真正的折扣应由自己的交易日志验证，而不是永久照抄。

## 八、0DTE 的特殊风险

0DTE 不只是“更便宜的方向工具”。临近到期时，Delta、Gamma、Theta 与隐含波动率变化会使权利金对价格和时间极为敏感。

实盘需要额外遵守：

1. 止损首先绑定 SPX 结构，同时设置期权端的灾难保护。
2. 不把全部权利金都视为默认可亏金额。
3. 接近目标节点先兑现一部分，不等“刚好打满目标”。
4. 快速盈利后，可用新的结构低点或权利金高点回撤管理剩余仓位，但追踪止盈不能替代初始止损。
5. 流动性下降、买卖价差扩大或事件临近时，仓位必须折扣。
6. 不因“最大亏损有限”而把多个有限风险组合叠加成过大的账户风险。

FINRA 的日内交易风险披露强调，日内交易可能产生快速而重大的亏损，波动或市场异常时也可能难以及时以合理价格退出。[[FINRA Rule 2270](https://www.finra.org/rules-guidance/rulebooks/finra-rules/2270)]

## 九、每日风险熔断

风险熔断的作用不是预测下一笔会不会赢，而是在执行质量下降时保护账户。

```text
达到 -1R：暂停，复核两笔交易是否属于同一错误
达到 -2R：当天停止主动交易
连续两笔完整止损：停止寻找“翻本单”
出现数据故障或异常滑点：降级为观察模式
违反一次预设规则：下一笔风险至少减半
```

“今天必须赚回来”不是交易目标。每日风险预算一旦用完，继续交易相当于在没有计划的情况下重新提高风险上限。

## 十、盈亏比必须基于结构，而不是愿望

```text
计划盈亏比 = 第一合理目标距离 ÷ 结构止损距离
```

例如：

```text
入场：100
结构止损：99
第一目标：102
计划盈亏比 = 2 ÷ 1 = 2R
```

如果下一个关键节点只提供 0.7R 的空间，即使方向判断正确，也不一定值得建仓。不能为了得到漂亮的 2R，把目标越过真实阻力随意画远。

分批止盈后，应记录实际实现的 R，而不是只记录最高浮盈：

```text
一半仓位在 +1R 退出
一半仓位在 +2R 退出
实际结果 = 0.5 × 1R + 0.5 × 2R = +1.5R
```

## 十二、本章总结

止损与仓位设计可以浓缩成五句话：

```text
止损放在逻辑失效处，不放在心理舒服处。
先确定失效距离，再倒推允许仓位。
分批建仓可以增加证据，但不能突破原定 1R。
最小合约超过预算时，换工具或不交易。
达到每日熔断后停止，不用下一笔交易修复情绪。
```

最终流程是：

```text
位置与触发
→ 定义结构失效
→ 计算 1R
→ 倒推仓位
→ 检查目标与盈亏比
→ 执行并按结构管理
→ 时间或环境失效时主动退出
```
