# 日内交易基础课：科学建仓

这套课程不以任何一位交易者的个人风格为中心，而是把多种成熟的日内操作经验整理成一套可以观察、执行和复盘的建仓流程。

> 核心目标：把“我觉得会涨跌”，改写成“价格在什么位置、出现什么行为、在哪里证明判断错误”。

## 统一建仓框架

```text
位置 → 触发 → 确认 → 失效 → 目标
```

| 环节 | 要回答的问题 |
| --- | --- |
| 位置 | 价格是否到达事先标记的关键区域？ |
| 触发 | 突破、拒绝、收回或结构改变是否已经出现？ |
| 确认 | 市场最终接受在关键位的哪一侧？ |
| 失效 | 什么价格行为能够明确证明判断错误？ |
| 目标 | 下一结构位能否提供足够的盈亏比？ |

## 建议学习顺序

先用第一章建立统一语言，再用第二章练习盘前画图与开盘分类，最后用第三章反复识别“突破被接受”和“突破被拒绝”。

每次复盘只记录六项：

```text
关键位置：
触发行为：
确认行为：
结构失效：
第一目标：
实际结果与偏差：
```

后续章节将继续补充止损与仓位、持仓与退出、复盘与迭代。
