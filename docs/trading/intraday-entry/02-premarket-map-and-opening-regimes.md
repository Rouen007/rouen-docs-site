# 第二章：盘前地图与开盘环境分类

> 盘前的任务不是预测全天涨跌，而是提前定义：价格到了哪里需要观察，出现什么行为才允许交易。

## 一、盘前只做一件事：建立地图

盘前分析的重点不是给出一个方向，而是把可能产生反应的区域标出来。

建议优先标记：

- 前一交易日高点和低点
- 盘前高点和低点
- 开盘价
- 第一根 10 分钟 K 线的高点和低点
- VWAP 可能所在的位置
- 主要支撑、阻力和流动性集中区
- 重要整数位
- 当天的新闻、数据和财报风险

可以把这些位置分成三层：

~~~text
执行区：距离当前价格较近，今天可能直接测试
目标区：距离适中，可作为第一目标或第二目标
背景区：距离较远，只用于理解大方向，不作为直接入场理由
~~~

盘前不需要把所有历史线都放进图里。实际执行时，通常只保留最近的 2—3 个上方和下方关键位置，避免图表过度复杂。

## 二、不同工具承担不同任务

指标和图表工具不应被混合成一个“预测按钮”。每个工具应该有明确职责：

| 工具 | 主要问题 | 不应该承担的任务 |
| --- | --- | --- |
| GEX、墙位、流动性区 | 价格可能在哪里减速、吸附或反应？ | 直接预测涨跌方向 |
| 前高、前低、ORB | 当前价格是否正在突破重要结构？ | 突破第一根直接追单 |
| VWAP | 当前持仓是否处在相对有利的一侧？ | 把它当成必然反转线 |
| 成交量、资金流 | 这次突破是否得到持续参与？ | 只看单笔大单就跟随 |
| 1 分钟 K 线 | 如何执行和管理仓位？ | 单独预测全天走势 |
| 3 分钟、5 分钟 K 线 | 当前结构是否连续？ | 忽略大级别位置 |

科学的使用顺序是：

~~~text
位置 → 价格结构 → 回踩/拒绝 → VWAP → 量价与资金流 → 辅助指标
~~~

后面的指标只能提高或降低把握，不能推翻前面的价格事实。

## 三、开盘阶段：先观察，再分类

开盘最容易出现噪音、扫损和情绪化突破。前 5—10 分钟的主要任务是观察，而不是急于完成第一笔交易。

开盘需要记录：

1. 第一段价格区间的高点和低点。
2. 价格是否快速离开开盘区间。
3. 突破后是否回到原区间。
4. 高点和低点是否开始持续推进。
5. VWAP 是否有明显斜率，价格是否持续位于同一侧。
6. 关键位置附近是否出现拒绝或接受。

开盘第一根 10 分钟 K 线可以提供早盘边界，但它不是追单信号：

~~~text
突破 ORB → 不追第一根 → 等回踩 → 回踩在区间外 → 再次启动
~~~

## 四、三种主要市场环境

### 1. 趋势环境

常见特征：

- 高点和低点持续向同一方向推进
- 多头回踩不跌回旧区间，空头反抽不上旧区间
- 价格大部分时间位于 VWAP 同一侧
- 回踩浅，重新启动速度较快
- 关键位置被突破后，市场愿意在新区间停留

趋势环境的交易方式：

~~~text
只做顺势回踩
止损放在回踩结构外
目标看下一个关键位置
到目标前分批兑现
~~~

趋势日最容易犯的错误，是把正常回踩误判成反转，或者在已经远离结构后追单。

### 2. 震荡或吸附环境

常见特征：

- 开盘价或 VWAP 被反复穿越
- 上下关键位置都出现反应，但没有持续突破
- 高点和低点没有明显推进
- 价格围绕中轴反复波动
- 关键节点被多次测试却没有形成接受

震荡环境的交易方式：

~~~text
中间区域不做
上沿等待拒绝
下沿等待拒绝
目标看回中轴或对侧边界
突破并回踩站稳后，才切换成趋势打法
~~~

震荡日最重要的不是增加指标，而是减少交易次数。中间位置没有足够的风险收益优势。

### 3. 失败突破或反转环境

常见特征：

- 价格先突破开盘区间或前高/前低
- 突破后快速回到原区间
- 原来的突破方向无法再次站回关键位
- 随后出现反抽失败或回踩失败
- 结构开始向反方向推进

失败突破的交易方式：

~~~text
第一次冲出：只观察
快速回到原区间：突破降级
反抽/回踩再次失败：反向建仓
止损放在失败突破的极值外
目标看中轴、开盘区间另一侧或下一个节点
~~~

不能因为一次长影线就立即反手。长影线只是拒绝的可能，后续收回和再次测试失败才是确认。

## 五、用决策树分类，而不是凭感觉分类

~~~mermaid
flowchart TD
    A[价格到达关键位置] --> B{是否有效突破或收回?}
    B -- 否 --> C[继续观察；中间区域不交易]
    B -- 是 --> D{回踩后是否留在新区间?}
    D -- 是 --> E[趋势候选：顺势回踩建仓]
    D -- 否 --> F{是否快速回到原区间?}
    F -- 是 --> G[失败突破候选：等待反抽/回踩失败]
    F -- 否 --> C
~~~

这棵决策树的核心不是预测，而是根据价格最终“接受在哪个区间”来决定打法。

## 六、如何处理不确定的开盘

如果开盘后出现以下情况，应当降低频率和仓位：

- 价格在两个关键位置之间来回穿越
- VWAP 几乎没有斜率
- 突破没有后续跟随
- 价格和主要指数不同步
- 事件风险即将发布
- 只能找到方向，找不到明确失效点

这时最合理的选择是：

~~~text
等待价格到达边界
→ 等待拒绝或接受
→ 重新计算目标与失效
→ 小仓或观望
~~~

没有清晰环境时，空仓不是错过机会，而是避免在低质量区域消耗风险预算。

## 本章总结

盘前地图负责回答“在哪里观察”，开盘结构负责回答“今天是什么环境”，回踩或失败测试负责回答“是否可以建仓”。

~~~text
盘前画位置
→ 开盘看结构
→ 分类市场环境
→ 选择对应打法
→ 等待确认后建仓
~~~

最重要的纪律是：

> 不要用趋势打法交易震荡，也不要用抄底摸顶的方式交易趋势。
