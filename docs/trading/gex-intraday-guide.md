# 用 GEX 数据做 SPX 日内交易：回测驱动的操作手册

> 本文的写法是"数据在前、操作在后"：每一条操作规则都必须先有回测统计支撑，没有数据支撑的直觉不进这份文档。
> 回测样本：SPX 分钟级 GEX 结构数据 **71 个交易日**（2026-04-07 ~ 07-17），其中做市商仓位光圈（黄紫线）数据 46 日；另有 TSLA 27 日作为单票对照组。样本随交易日持续滚动扩充，统计数字以数据集终稿为准。
> 本文是个人研究笔记，不构成投资建议。

---

## 一、方法总纲

### 1.1 为什么 GEX 有效（唯一需要懂的原理）

做市商卖出期权后必须对冲，对冲量随价格变化（Gamma）。这个行为是**数学强制，不是市场心理**：

```
做市商净正 Gamma → 涨了卖、跌了买 → 高抛低吸 → 压制波动（震荡、pin、磨）
做市商净负 Gamma → 涨了追、跌了杀 → 追涨杀跌 → 放大波动（趋势、加速）
```

所以 GEX 数据能提前告诉你两件事：**今天大概会怎么动（环境）**、**哪些价位会有对冲行为发生（节点）**。它回答不了的是方向——方向永远由价格行为确认。

### 1.2 指标术语表

| 指标 | 定义 | 一句话用法 |
|---|---|---|
| **TG**（Total GEX） | 全市场净 Gamma 敞口 | 正=震荡游戏，负=动量游戏 |
| **King** | GEX 绝对值最大的行权价 | 当日最强磁力位 |
| **CW / PW** | Call Wall / Put Wall（持仓墙） | 天然阻力/支撑，框出区间 |
| **FLIP** | 正负 Gamma 分界价位 | 上方偏稳，下方偏动量 |
| **Air Pocket** | 节点间的 GEX 真空带 | 价格加速穿越——是路不是站 |
| **黄线** | 正 Gamma 仓位光圈 | 减速带/磁铁 |
| **紫线** | 负 Gamma 仓位光圈 | 触发器，**不是支撑阻力** |
| **fade** | 逢冲反做（冲高做空/杀跌做多），与"追"相反 | SPX 早盘主力打法（[M2·冲动仅 46% 延续](#m2)）；动量单票禁用 |
| **squeeze** | 挤压——假突破后被困仓位止损离场推动的反向急行情 | 紫线触发失败后的标准后续（[P1·反向 75%/+5.5 点](#p1)） |

### 1.3 三条永久原则

```
1. GEX 给地图，价格行为给确认 —— 两者对齐才入场，冲突时等待
2. 节点是观察区，不是买卖按钮 —— 止损设节点外一格
3. 结构剧变 = 暂停 —— King 跳位、结构分钟级洗牌时不交易
```

## 二、回测结论：什么被验证了、什么被证伪了

> 编号体系：S = 日级规则，M = 早盘规则，N = 节点规则，P = 黄紫线规则。每条标注样本与强度，小样本规则明确标注仓位折扣。
> **全文所有规则编号都是可点击链接**（如 [M2·冲动仅 46% 延续](#m2)），点击跳回本章定义处；链接文字自带一句话释义，不点也能读。

### 2.1 日级规则（盘前定基调）

#### S1 · FLIP 位置 = 最强单一信号 ⭐⭐⭐ {#s1}

| 全天价格相对 FLIP | 样本 | 日均收益 | 收跌概率 | 日均振幅 |
|---|---|---|---|---|
| 上方 | 34 日 | **+32.7 点** | **3%** | 59.5 |
| 下方 | 23 日 | -23.7 点 | **70%** | 81.2 |
| 来回穿越 | 14 日 | -5.2 点 | 64% | 81.6 |

→ **操作**：现价在 FLIP 上方只找做多理由，下方只找做空理由；盘中来回穿越的日子降仓或不做（它们既难又振幅最大）。换边判定需持续 ≥30 分钟，单次穿越是噪音（[N3·FLIP 日内漂移中位 106 点](#n3)）。

⚠️ **开盘时刻的正确用法（防后视镜误读）**：S1 的分组条件是"全天待在哪一半"——这只有收盘后才知道，且有循环成分（一天没跌破 FLIP 本身就说明这天强，位置既是原因也是结果）。**开盘那一刻可用的版本是 [M3](#m3)：10:00 在上方 → 后续上涨 61%，下方 → 43%**——有边际优势但远小于 97/30，所以只配当方向开关，不配当入场理由。S1 大数字的兑现方式是**状态机 + 持续监控**，不是开盘一次性下注：

```
开盘在上方 → 进入"偏多模式"（假设，不是结论）
每次回踩 FLIP = 检验点：守住 → 假设续期；跌破持续 ≥30 分钟 → 换边
开始来回穿 → 降级为最差组（64% 收跌）→ 降仓或不做
```

反例提醒：曾有交易日全天 100% 在 FLIP 下方仍收涨 +10.3（深负跳空的对冲耗尽反弹，见 [S4·深负方向不对称](#s4)）——事后分组也只是 base rate，不是铁律。

**距 FLIP 的距离 = 缓冲垫，不是信号**。距离决定的是"今天退化成来回穿越日"的风险等级：

```
距 FLIP >20-30 点 → bias 可信度高，正常仓
距 FLIP <10 点   → 一个普通回踩就可能把今天甩进最差组 → 降仓，FLIP 破位写进失效条件
正骑在 FLIP 上   → 不做
```

#### S2 · 负转正 = 做多日 ⭐⭐⭐（小样本，半仓）{#s2}

TG 连续为负的时期结束、回正的那一天：11/11 收涨；其中"开盘即正"（9:30 即可识别）的 7/7 全胜。
→ **操作**：连续负 TG 期间每天开盘查 TG 符号，转正当天默认做多剧本，仓位减半（n=7 太小）。

这条同时是"把事后统计翻译成事前可用"的方法论范本：原始发现（回正日 11/11）里"哪天算回正日"有事后成分，于是切出 9:30 即可确认的"开盘即正"子集——样本从 11 缩到 7、结论降级、强制半仓。**翻译的代价永远是样本变小、数字变谦虚；不肯付这个代价的规则都在偷看答案。**

#### S3 · 负 GEX = 动量环境 {#s3}

负 GEX 日均振幅 85.4 点 vs 正 GEX 62.5 点（+37%，n=26/45）。
→ **操作**：止损与目标按环境缩放——负 GEX 日止损给更宽、目标看更远；正 GEX 日反之。

#### S4 · 深负 TG（<-1000B）方向不对称 {#s4}

开盘杀跌→顺势别接；开盘冲高→fade。两个豁免：
- 闪崩次日：修复反弹，禁止顺势做空
- **跳空子类型**：≥50 点 gap down + 首 30 分不创新低 + 早盘持续正收益 → 杀跌动作已被跳空提前走完，剩下的是对冲耗尽机械反弹，**不做空反手偏多**（实测案例：跳空 -87 点、TG -4119B、全天在 FLIP 下方，仍收 +10.3）

#### S5 · CW<PW 倒挂 —— 已证伪，停用 ⛔ {#s5}

早期 40 日样本显示倒挂日次日平均 +21.8，看似强信号；71 日重算后：倒挂日次日 +15.3 vs 正常日 +10.9，净 edge 仅 4.4 点且方向命中率反而更差。旧结论是单边上涨行情的样本偏差。
→ **教训**：这条留在文档里是故意的——它示范了本方法论的核心纪律：**规则必须能被更大的样本证伪，被证伪就停用**，不跟数据讨价还价。

### 2.2 早盘规则（9:30-12:00 ET，主战场）

| # | 结论 | 数据 | 操作含义 |
|---|---|---|---|
| <span id="m1"></span>M1 | 早盘含全天 77% 振幅 | 71 日 | 只做早盘不吃亏，12:00 清仓不留午 |
| <span id="m2"></span>M2 | 开盘冲动 fade 优势 | 首 30 分 ±15 点后仅 **46%** 延续；冲高后续均 -7.3，杀跌后续均 +11.7 | 统计优势在逢冲反做；杀跌后做多比冲高后做空肉更大 |
| <span id="m3"></span>M3 | 10:00 FLIP 过滤 | 上方→后续上涨 61%，下方→43% | 只当方向开关，不当入场信号 |
| <span id="m4"></span>M4 | 前日 King 磁吸 | 开盘距 <15 点 → **85%** 早盘触及（中位第 18 分钟） | 它是早盘第一磁力目标/止盈锚；距离 >40 点从图上划掉 |
| <span id="m5"></span>M5 | King 首小时迁移 = 早盘方向 | 上移 ≥10 → 早盘均 +20；下移 ≥10 → 均 -29 | King 追着价格跑的方向就是早盘 bias |
| <span id="m6"></span>M6 | 11:00 后只顺势 | 顺势 67% 延续；逆势 fade 期望 -5/单 | 11:00 后不开反向新单 |
| <span id="m7"></span>M7 | 时间效率衰减 | 早盘极值中位 11:28 走完；11:00 后每分钟效率掉 35% | 收工线 11:30，12:00 硬性底线 |

**M4 使用警告——路径 ≠ 方向**：King 在现价下方 12 点、85% 会被触及，说的是"价格大概率往下够一下"（**路径**），不是"今天偏空"（**方向**）。方向由 FLIP 位置定（[S1](#s1)/[M3](#m3) 的事），路径由磁吸定（[M4](#m4) 的事）。两者叠加才能读出操作：环境偏多 + 磁吸目标在下方 = 下探 King 是**低吸位**（到位等 K 线拒绝做多回中轴），不是做空机会——在 3% 收跌环境里追空 12 点的磁吸路径，方向、赔率、磁吸反弹三头逆风。凡想说"方向"时先自问：这是环境（[S1·FLIP 定方向](#s1)）的事，还是路径（[M4·King 磁吸](#m4)）的事？

### 2.3 节点规则

| # | 结论 | 数据 | 操作含义 |
|---|---|---|---|
| <span id="n1"></span>N1 | 收盘 pin King | 收盘距 EOD King 中位 **2.6 点**、70% ≤5 点、97% ≤10 点 | 全数据集最稳规则；尾盘结构单的理论基础（本框架只做早盘，故仅用于理解磁吸） |
| <span id="n2"></span>N2 | 盘前快照保质期 | King 到 10:00 漂移中位 0 点、68% 稳在 ±5；到 12:00 只剩 47% | 底图对早盘前半段够用；**King 早漂 = 趋势日信号**，不是数据错了 |
| <span id="n3"></span>N3 | FLIP 动态漂移 | 日内漂移中位 106 点；78% 漂移方向与当日涨跌同向 | FLIP 追价 = 趋势确认；TG <±500B 时 FLIP 不稳定，退回用 King/Wall |
| <span id="n4"></span>N4 | 底图作废条件 | 夜盘 ≥20 点穿越多合一节点 | 旧节点只留"汇合区"参考，必须等盘前新快照 |

### 2.4 黄紫线规则（46 日）

#### P1 · 紫线触发失败 = 全数据集质量最高的入场 ⭐ {#p1}

强紫线穿越事件 n=115：

```
65% 穿过后延续（触发成功）→ 顺向 15 分钟 61% / +2.3 点 —— 顺势信号，禁止 fade
35% 在 2-4 根 K 内收回（触发失败）→ 反向 15 分钟 75% / +5.5 点 —— 最高质量反向入场
```

→ **操作**：紫线不是支撑阻力，是真假突破检测器。它只回答一个问题：这次突破是真扩张还是假触发。

#### P2 · 黄线裸首触无 edge（无优势）{#p2}

黄线首触后仅 28% 出现有效反弹，亮度分级不改善此数字。
→ **操作**：**"碰线就接"没有优势。K 线拒绝（长影/假突破收回）才是入场理由本身，不是过滤器。** 这条推翻了"强节点碰到就反弹"的直觉，是全文档最值钱的负结论之一。

<span id="p3"></span>**P3 · 黄紫线预热 60 分钟**：当日仓位光圈从零累积，前 60 分钟置信度低 → 9:30-10:30 用隔夜骨架（King/CW/PW/FLIP）为主，10:30 后光圈转主图。

<span id="p4"></span>**P4 · 亮度是帧内相对值**：只用于当前画面排序，不能跨日比较"今天比昨天亮"。

### 2.5 对照组：这套规则不通用于单票 ⚠️

TSLA 27 日镜像回测：开盘冲动 **78%** 延续（SPX 仅 46%）→ 单票动量股禁 fade 只顺势；黄线是目标位不是入场位；收盘不 pin（0.23% vs SPX 0.04%）。**本文档全部 fade 逻辑仅适用于 SPX/SPY。** 唯一通用铁律：裸首触无 edge，必须等 K 线拒绝。

## 三、每日操作流程（把规则串成动作）

### 3.1 盘前（9:00-9:25 ET）

```
□ 记 TG 符号与量级        → [S2·负转正做多日？](#s2) / [S3·振幅预算](#s3) / [S4·深负剧本](#s4)
□ 标 King 到价格图        → [M4·距开盘价 <15 点 = 早盘第一磁力目标](#m4)
□ 标 CW / PW / FLIP       → [S1·今天站哪一半](#s1) / 区间上下沿
□ 标开盘价 ±30 观察位     → 分层：100 点大地图 / 50 点剧本 / 30 点观察 / 10-15 点执行
□ 查夜盘有无 ≥20 点破位   → [N4·底图作废？等新快照](#n4)
□ 写下环境预判 + 三个剧本（站稳 A 看 B / A 拒绝看 C / 打穿反抽不过看 D）
没画完节点不开盘 —— 这条凌驾于一切。
```

### 3.2 开盘环境判定（9:30-9:45，只看不做）

| 检查项 | 区间日 ✅ | 趋势日 ⚠️ | 震荡日 🛑 |
|---|---|---|---|
| TG | 正，稳 | 负或快速翻负 | 正负摇摆 |
| King | 稳定不动 | 单向迁移（[M5·迁移方向=早盘方向](#m5) / [N2·早漂=趋势日信号](#n2)） | 来回跳 |
| FLIP | 价格在上方 | 被破不回头 | 位置不稳 |
| 打法 | 节点边界反转 | 顺势+回踩加仓 | 小仓位或不做 |

**不确定时默认震荡日**——用"不交易"应对任何环境最多少赚，不会大亏。

9:45 第一次刷新：King 原位（±5）= 底图有效；跳 ≥2 档 = 趋势日信号，只顺势不 fade。

三个使用要点：

- **King 的判据是方向一致性，不是动没动**：趋势日=单向迁移（连续刷新朝同一边），震荡日=来回跳。看到 King 动，先盯两三次刷新确认方向再定性。
- **默认震荡日的原因是代价不对称**：区间打法进趋势日被甩、趋势打法进震荡日被反复扫，只有"小仓/不做"用错环境时仅仅少赚——分不清时选错误代价最小的那个。
- **盘前预案的价值不在执行它，而在 9:45 有个明确的东西可以被推翻**：King 跳档、底图作废本身就是当天最重要的信号（[N2·King 早漂 = 趋势日自我暴露](#n2)）。预案作废时连锚都没了——守着旧 King 等低吸，等的是一个已不存在的节点。

### 3.3 三个主力 Setup（参数全部来自回测）

**Setup A · 开盘冲动 fade**（依据 [M2·冲动仅 46% 延续](#m2)）

```
第一步 豁免检查（依据 [S4·深负方向不对称](#s4)）：
  □ 前一日闪崩/大跌？→ 今天冲高是修复反弹，禁 fade
  □ 今晨 TG 强正（>+1000B）且昨日也正？→ 趋势冲动，顺势不 fade
第二步 看冲动撞上什么线：
  撞黄线 + K 线拒绝（长影/收回）→ 标准 fade ✅
  撞黄线直接穿透 → 不追不 fade，黄线角色反转，等回踩再顺势
  撞紫线穿过延续 → 触发成功 = 顺势信号，⛔ 禁 fade
  撞紫线 2-4 根 K 收回 → 触发失败 = 反向挤压（[P1·反向 75%/+5.5](#p1)）✅ 最高质量
  没撞线（进 Air Pocket）→ 等撞到下一条线再评估
参数：SL 冲动极值外 10-15 点 / TP1 开盘价 / TP2 King（前 5-10 点先收）
优先级：杀跌后做多（+11.7）> 冲高后做空（-7.3）
```

**Setup B · 强节点首触反向**（依据 [P2·裸首触仅 28%，确认不是可选项](#p2)）

```
对象：只做亮黄线/强 King 首触；紫线没有"首触反弹"属性
必要条件：首触 → 等 K 线拒绝 → 才反向入场（裸接只有 28% 胜率）
加分项：黄线与 King/Wall 同价位共振 / 到对面目标 ≥15-20 点
纪律：同一节点第 2 次触碰只减仓不加仓；第 3 次反着想——它要破了
SL：节点外一格（10-15 点）/ TP：下一节点，前 5-10 点先收
```

**Setup C · 方向过滤器**（依据 [M3·FLIP 只当开关](#m3) + [P1·紫线触发状态](#p1)）

```
主开关 FLIP（10:00 定调）：上方只做多向 / 下方只做空向 / 贴着不带偏见
副开关 紫线：某方向触发已成功 → 反向 setup 全部作废
          两侧都触发失败 → 震荡日确认，只做节点边界往返
```

### 3.4 执行八原则（凌驾于所有 setup）

```
1. 先画节点和区间，没画完不开盘
2. 开盘看价格往哪个大节点走 —— 方向是价格自己选的路，不预判
3. 离节点 <10 点不追单
4. 突破不追第一根，等回踩守住才上
5. 节点测试失败就看反向 —— 反向剧本立刻激活
6. 两节点中间的无人区只看不做
7. 到节点分批止盈，前 5-10 点先收一半
8. flow / 跨指数同步只做确认加分，不单独构成入场理由
```

### 3.5 时段纪律

```
9:30-9:45    ⛔ 只看不做
9:45-11:30   ✅ 核心窗口
10:00 ±10min ⛔ 经济数据雷区，唯一固定停手点
11:00 后     只管理持仓 + 顺势，不开反向新单（[M6·逆势 fade 期望为负](#m6)）
11:30        默认收工（[M7·极值中位 11:28 已走完](#m7)）
12:00        硬性清仓，无论如何

提前收工：第一单全须全尾 → 当天结束；10:45 前无 setup → 磨面日，收工
频率上限：每天最多 2 单，第二单只在第一单盈利后允许
期权结构：单腿 0DTE 或垂直 debit 价差；不做 IC/IB（pin 策略要拿到 16:00，与 12:00 清仓冲突）
```

### 3.6 出场与风险

```
到次级节点收一部分 → 到大节点再收一部分 → 目标前 5-10 点不贪满
入场后 2-4 根 K 没按预期离开节点 → 主动降仓或退出
+20% 利润 → 止损拉到保本位；卖飞是好结果
趋势仓可用期权价自高点回撤 10-15% 追踪止盈，但不替代结构止损
结构失效（关键位失守）→ 剩余仓位全走，不犹豫
```

**不做清单**（任何一条命中就空仓观察）：

```
价格在两节点中间 / 到下一节点不足 15-20 点 / 刚触节点没反应

```
