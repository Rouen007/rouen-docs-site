# 如何在个人知识库（rouen-docs-site）添加新内容

这个知识库基于强大的 VitePress 构建，添加和修改内容非常简单。你只需要记住以下核心的 **“三步走”** 原则。

---

## 核心三步走

1. **写一篇文章**：把 `.md` 文件放到对应的分类文件夹里。
2. **挂到菜单上**：在配置文件里告诉侧边栏有这篇文章。
3. **推送到云端**：用 Git 上传，网站会自动更新。

---

## 详细操作指南

### 第一步：创建或放入你的 Markdown 文件

打开你的本地文件夹 `/Users/rouen/Documents/rouen-docs-site/docs`，你会看到我们分好的频道结构。

假设你写了一篇关于“期权基础”的新笔记（`options-basic.md`），你想把它放进**交易频道**里。
你只需要把文件拖进去即可，例如路径变为：
`/docs/trading/options-basic.md`

*(提示：如果你有配图，可以在同级目录新建一个 `assets` 文件夹，例如把图片放在 `/docs/trading/assets/` 下，然后在文档里用相对路径 `![描述](./assets/图.png)` 引用即可)*

### 第二步：在左侧边栏 (Sidebar) 增加这篇文档的链接

虽然文件放进去了，但网站的侧边栏还不知道它的存在。我们需要配置一下。

1. 打开配置文件：`/docs/.vitepress/config.mjs`
2. 找到 `sidebar` 配置块下的 `/trading/` 部分。
3. 把你的新文章加到 `items` 列表里。

**修改前：**
```javascript
      '/trading/': [
        {
          text: '导读',
          items: [
            { text: '交易主页', link: '/trading/' }
          ]
        },
        {
          text: '专题研究',
          items: [
            { text: 'Heatmap 五层框架', link: '/trading/heatmap/five-layer' }
            // 你将在这里添加新内容
          ]
        }
      ],
```

**修改后（增加了一行）：**
```javascript
      '/trading/': [
        {
          text: '导读',
          items: [
            { text: '交易主页', link: '/trading/' }
          ]
        },
        {
          text: '专题研究',
          items: [
            { text: '期权基础入门', link: '/trading/options-basic' }, // 新增的这一行！(注意结尾不需要写 .md)
            { text: 'Heatmap 五层框架', link: '/trading/heatmap/five-layer' }
          ]
        }
      ],
```

*(注意：`link` 里面的路径**不要**带 `.md` 后缀。)*

### 第三步：提交并发布

在终端（Terminal）进入网站根目录，执行标准的 Git 推送三连：

```bash
cd ~/Documents/rouen-docs-site

# 1. 暂存所有改动
git add .

# 2. 写个简单的备注
git commit -m "Add new article: options-basic"

# 3. 推送到 GitHub
git push
```

**大功告成！** 只要你执行了 `git push`，Cloudflare Pages 或 GitHub Pages 就会在云端自动抓取最新代码，1 分钟后你的网站就更新啦。

## 💡 进阶小贴士 (Tips)

### 如何在本地边写边看效果？
每次写长文章时，你可以先在本地开服，边写边实时预览，不用急着传到网上。
1. 在终端里运行：
   ```bash
   cd ~/Documents/rouen-docs-site
   npm run docs:dev
   ```
2. 浏览器打开 `http://localhost:5173`。
3. 只要你保存 Markdown 文件，浏览器里的画面会**瞬间自动刷新**，体验极佳。写完满意了，再执行上面的第三步推送。

### 想要新增一个全新的顶层大频道（比如 "读书笔记"）？
1. 在 `/docs/` 下新建一个文件夹 `books/`，在里面放一个 `index.md`。
2. 打开 `config.mjs`，在 `nav: [...]` 里加一行 `{ text: '读书 📚', link: '/books/' }`。
3. 在 `sidebar: {...}` 里模仿 `/trading/` 的格式，照猫画虎增加一块 `'/books/': [...]` 的配置即可。
